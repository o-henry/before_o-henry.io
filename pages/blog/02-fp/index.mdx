## Imperative 와 Declarative

명령형 프로그래밍은 결과를 위해 어떻게(How) 할지 지시 하는 과정이 포함되는데 반해, 함수형 프로그래밍은 해당 하는 것이 무엇(What)인지를 표현합니다.

<br />

## Pure Function

순수 함수는 사이드 이펙트가 없고 같은 Input에 항상 같은 Output을 반환하는 함수를 의미합니다.

<br />

## Closure

클로저는 내부 함수가 외부 함수의 Context(맥락)에 접근할 수 있는 것을 말하는데, 해당 함수가 호출, 종료 되어 제거 되어도 계속 접근할 수 있습니다.

<br />

## Higher Order Function

함수형 프로그래밍을 제공하는 언어는 Higher Order Function 기능을 제공하는데, 이는 함수를 인자로도, 값으로도 사용할 수 있다는 말입니다.

<br />

## Curry

함수형 프로그래밍은, 함수의 조합으로 구현되는데, 이를 위해서는 Curry가 중요합니다. Curry를 통해 Compose를 구현합니다.

<br />

## Composition

다양한 함수 조합을 위하여 가장 기본이 되는 기능으로, 모든 기능을 레고 조각 처럼 함수들을 조합하여 구현합니다.
